Angular devDependencies

This is a list of all the required development dependencies to run Angular

  • concurrently

    This allows to run commands at the same time.

  • lite-server

    Node server that serves a web app. Watch html, css and javascript files with ease.

  • typescript

    A language for application-scale Javascript

  • canonical-path

    This utility simple wraps the built-in path library such that it only ever returns path segments separated by forward slashes (/).

  • tslint

    Linter for TypeScript

  • loadash

    For iterating arrays, objects and string. Manipulating and testing values. Creating composite functions

  • jasmin-core

    A javascript testing framework

  • karma

    Allows to execute javascript code in multiple real browsers.

  • karma-cli

    N/A

  • karma-jasmine

    Adapter for the Jasmine testing framework

  • karma-jasmine-html-reporter

    Reporter that shows result at debug.html page.

  • protractor

    End to end test framework for Angular and AngularJS applications

  • rimraf

    Unix command for rm -rf for node.

  • @types/node

    Type definitions for Node.js

  • @types/jasmine

    Type definitions for jasmine
